## var let const

### summary
1. immutable: const 
1. mutable: let
1. never use var
> const > let > var

#### var
1. scope: `global or function`
1. re-declare
1. use it before declare , initialize with `undefined` 

#### let
1. scope: `block`
1. can not re-declare
1. can not use it before declare `ReferenceError`

### const
1. scope: `block`
1. can not change
